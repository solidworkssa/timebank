# TimeBank

Time-based skill exchange platform.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/TimeBank.sol`
- Stacks: `contracts/stacks/contracts/timebank.clar`

## Setup
```bash
pnpm install
pnpm dev
```
